Easier integration process

With CloudOrbit we've made the form integration process as easy as possible. Recent form submissions are displayed at the top of the dashboard, so that you can quickly view new submissions without them getting missed. To keep your dashboard clean and fast, once you've viewed new submissions we'll hide the submission, but of course the full submission record can be accessed elsewhere.

Full management experience

We've made it so that you can quickly add multiple forms per website, and all your forms are clearly seperated so you can view submissions for each form. Also, add multiple websites to your dashboard giving you a full management experience.

How it Works

// Add this line to the top of your HTMl page
action = "www.appolo.com/api/forms/8aeb" method ="post"
// Add record type to your form
input type = "name" name = "record[first_name]"
input type = "email" name = "record[email]"
input type = "message" name = "record[message]"

No complex code, just basic HTML

  • Create a form using the 'forms' section of your dashboard and add the input fields your form contains.
  • We will give you a html form template if you need, or you can use an existing form already set up on your website.
  • For prebuilt forms, we'll give you a small snippet of code to place into the head of your html page
  • Add the input fields to your html page. Done! You're ready to start receiving submissions
